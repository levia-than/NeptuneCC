# Docs

This folder contains short references for the current NeptuneCC flow.

- `pragmas.md`: supported `#pragma neptune` forms and expected structure.
- `glue.md`: generated C++ glue layout and buffer conventions.
- `overlap.md`: overlap regions (safe interior + boundary faces) and rewrite order.
- `schedule.md`: deterministic Halide schedule inference and knobs.
- `testing.md`: how to run lit and end-to-end cases.
- `experiments/`: tuning and benchmark notes.
